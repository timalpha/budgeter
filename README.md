## Budgeter
Budgeting webapp written in PHP using Laravel 5.3 framework.


#### Install
Clone the repo
```shell
git clone https://github.com/timalpha/budgeter.git
```

Install php dependencies
```shell
composer install
```

Install other dependencies
```shell
npm install
```

Migrate the database
```shell
php artisan migrate
```

Build assets
```shell
gulp
```
